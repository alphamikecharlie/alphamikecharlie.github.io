About

Inverse Reinforcement Learning Framework for Autonomous Decision-Making in an Ethical Dilemma

The principle of value alignment directs that autonomous decision- making should be aligned with the values of humans acting on a given scenario. Following this principle, we develop methods allow- ing an ethical exemplar to train an artificial agent so that the agent’s values will align with the exemplar’s. We then test our methods on an ethical dilemma that has no universally agreed upon solution. We propose an architecture that leverages Inverse Reinforcement Learning (IRL) to allow an agent to learn different ethical behaviors from human demonstrations, decoupling decision-making from action execution and utilizing a reward framework based on end states. We evaluate the proposed architecture on a home invasion scenario, training an agent to generate different behaviors when facing the threat of a home break-in by a hostile actor. Our tests show convergence over three different human-demonstrated poli- cies (hide, call the police, and kill the invader) for decision-making, with the agent first deciding what to do based on contextual input, and subsequently interacting with the environment with actions to execute its decision. The results demonstrate that the proposed architecture is capable of producing value-aligned ethical behavior and can, perhaps, serve as a template for similar scenarios that involve ethical dilemmas.
